# Gestoria 2.0 — Стратегический Дашборд

Конфиденциальный отчёт для запуска gestoría (бухгалтерского агентства) в Испании, ориентированной на S.L. компании с оборотом €1-10M и фокусом на ВЭД (импорт/экспорт).

## О проекте

Интерактивный дашборд с аналитикой рынка, юнит-экономикой, анализом конкурентов и технологическим стеком для MVP. Защищён паролем для конфиденциальности.

### Содержание отчёта

1. **Архитектура Рынка** — объём €4.2B, драйверы роста, психотипы клиентов
2. **Фискальные Ловушки** — критические риски IS, НДС, Intrastat
3. **Юнит-Экономика** — тарифы €400-950/мес, LTV/CAC, P&L прогноз
4. **Конкуренты** — матрица 6+ игроков с SLA, страховками, ценами
5. **Технологический Стек** — Lean MVP на Holded + Next.js + Supabase
6. **Юридическая Защита** — RC страховка, Cero Multas, гарантии компаний
7. **Стратегия & Запуск** — план на 4 фазы с метриками

## Технологии

- **Frontend:** React 19, Vite 7
- **Архитектура:** Модульные компоненты (chapters, ui, layout, tech, competitors)
- **Styling:** CSS Variables, адаптивная вёрстка, dark theme
- **Защита:** Пароль + localStorage сессия

## Быстрый старт

```bash
# Перейти в папку приложения
cd gestoria-report

# Установить зависимости
npm install

# Запустить dev сервер
npm run dev
```

Открыть http://localhost:5173

**Пароль для входа: `5555`**

## Ключевые компоненты

### Главы (chapters/)
| Компонент | Описание |
|-----------|----------|
| `MarketArchitecture` | Размер рынка, сегменты, психотипы клиентов |
| `FiscalTraps` | Риски IS, НДС, Intrastat с процентами штрафов |
| `UnitEconomics` | Тарифы, P&L прогноз, точка безубыточности |
| `StrategyLaunch` | 4-фазный план запуска, дифференциаторы, moats |
| `LegalProtection` | RC страховка, Cero Multas, контрактные клаузулы |

### UI компоненты
| Компонент | Описание |
|-----------|----------|
| `MetricCard` | Карточка с ключевой метрикой (value + subtext) |
| `BarChart` | Горизонтальный bar chart для сравнений |
| `PricingTable` | Таблица тарифов с сегментацией |
| `CompetitorTable` | Интерактивная матрица конкурентов с фильтрами |
