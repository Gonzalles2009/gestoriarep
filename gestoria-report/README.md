# Premium Gestoria Report Website

This project converts the analysis documents into a stylish, interactive "scrollytelling" website.

## How to Run
1.  `npm install`
2.  `npm run dev`
3.  Open `http://localhost:5173/`

## Structure
*   `src/data/content.js`: Contains the text content extracted from the analysis.
*   `src/App.jsx`: Main application logic.
*   `src/index.css`: Styling.
